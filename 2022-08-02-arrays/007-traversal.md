# All About Arrays - Traversal

```javascript
const myArray = [1, 2, 3];

for (let i = 0; i < myArray.length; i++) {
  console.log(myArray[i]);
}
```

```javascript
for (let item of myArray) {
  console.log(item);
}
```

```javascript
myArray.forEach(item => console.log(item));
```
